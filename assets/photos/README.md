# Photos folder

Drop your real photos here (jpg/png/webp), then update `CONFIG.photos` in `script.js`.

Example:
- `assets/photos/1.jpg`
- `assets/photos/2.jpg`
- `assets/photos/3.jpg`

Then in `script.js`:
```js
photos: [
  { src: "assets/photos/1.jpg", caption: "Us" },
  { src: "assets/photos/2.jpg", caption: "That day" },
  { src: "assets/photos/3.jpg", caption: "My favorite smile" },
],
```
